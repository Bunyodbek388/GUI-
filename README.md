# GUI Кітапхана Жүйесі

Бұл жоба — графикалық интерфейсі бар (GUI) кітапхана басқару жүйесі.  

Жүйе кітаптарды қосу, өшіру, іздеу және қолданушыларды басқаруға арналған.

## Мүмкіндіктері

- 📚 Кітап қосу

- ❌ Кітап өшіру

- 🔍 Кітап іздеу

- 👤 Қолданушыларды басқару

- 📝 Кітап тізімін көру

- 💾 Деректерді сақтау

## Қолданылған технологиялар

- Python

- Tkinter (GUI кітапханасы)

- SQLite (деректер базасы)
